# GeraCode — Projeto SEO

## Sobre o site
- **URL**: https://www.geracodigo.com.br/
- **O que faz**: Gerador gratuito de Código de Barras (EAN-13, EAN-8, Code 128, Code 39, UPC-A, ISBN), QR Code genérico e QR Code Pix para lojistas brasileiros.
- **USP principal**: 100% client-side — nenhum dado do usuário sai do navegador. Privacidade total.
- **Público-alvo**: Lojistas brasileiros, MEIs, e-commerces pequenos e médios, desenvolvedores.
- **Stack**: HTML/CSS/JS client-side, sem backend. Hospedado em domínio próprio.

## Prioridades SEO
1. **Schema markup** — SoftwareApplication, HowTo, FAQPage para Rich Results
2. **Core Web Vitals** — site client-side deve ter LCP < 1.5s, CLS < 0.1
3. **Content expansion** — blog/tutoriais para capturar long-tail (ex: "como gerar EAN-13 para Shopee")
4. **GEO/LLMO signals** — otimizar para aparecer em respostas de ChatGPT, Perplexity, Google AI Overviews
5. **Internal linking** — cross-linking entre ferramentas e conteúdos educacionais
